This is a [Next.js](https://nextjs.org) example client for Anytime's API.

## Running the server locally

Install dependencies

```bash
npm i
```

Then, start a local NextJS server

```bash
npm run dev
```

or

```bash
npm run build
npm run start
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
